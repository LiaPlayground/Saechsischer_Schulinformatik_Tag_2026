# WS2.03. LiaScript im Informatikunterricht

Online-Workshop anlässlich des [Sächsischen Schulinformatiktag 2026](https://events.sachsen.schule/event/p/sit26online), 18. März 2026.

## Inhalt

LiaScript ist eine Erweiterung von Markdown, die es ermöglicht, interaktive Lehrmaterialien als einfache Textdateien zu erstellen — ohne spezielle Software, ohne Server. Dieser 90-minütige Workshop zeigt, wie Informatiklehrkräfte LiaScript nutzen können, um Quizze, editierbare UML-Diagramme, Arduino-Simulationen und ausführbaren Python-Code direkt im Browser bereitzustellen.

Der Workshop folgt dem Prinzip **"Ein Dokument für alles"**: Die Präsentation selbst ist ein LiaScript-Kurs und damit gleichzeitig Anschauungsobjekt.

**Themen:**

- Interaktive Elemente erleben (Quiz, PlantUML, Arduino, Python)
- Kernkonzepte von LiaScript verstehen
- Live Hacking im LiveEditor
- Verbreitung: Von GitHub nach OPAL-Schule
